riid.anomaly module¶

This module contains an algorithm for detecting statistical anomalies from sequences of gamma spectra.

class riid.anomaly.PoissonNChannelEventDetector(long_term_duration: int = 120, short_term_duration: int = 1, pre_event_duration: int = 5, max_event_duration: float = 120, post_event_duration: int = 1.5, tolerable_false_alarms_per_day: float = 1.0, anomaly_threshold_update_interval: float = 60)¶

Bases: object

This class identifies sequences of anomalous spectra.

This algorithm assumes that the background environment will be consistent over known timescales. I.e., this algorithm is only intended for static detector, mobile source scenarios.

add_measurement(measurement_id: int, measurement: array | int, duration: float, verbose=True)¶

Adds the next measurement to the event detector.

Parameters:

  • measurement_id – a unique identifier for the measurement.

  • measurement (array-like) – a 1D array-like object containing one or more channels of measured data.

  • duration – a float representing the length of time over which the measurement was taken.

  • verbose – whether to print log messages.

Returns:

  • the channel-wise sum of all spectra that were part of the event
    • the channel-wise sum of all spectra in the background buffer,

      normalized to match the duration of the event

    • the duration (in seconds) of the event

    • list of measurement IDs that comprise the event

else if adding a spectrum does not conclude event, return None
